# Changelog 📝

All notable changes to **Student Study Buddy** will be documented in this file.

---

## [1.1.0] - 2026-06-13
### Added
- **Bring Your Own Key (BYOK)** cloud LLM support for **OpenAI** and **Google Gemini**.
- Dynamic model selection based on active provider.
- Conditional API key inputs in the sidebar with status indicators.
- Utility script `push_to_github.sh` for streamlined deployment.

---

## [1.0.0] - 2026-06-12
### Added
- Core backend system for document parsing (PDF, DOCX, TXT).
- FAISS vector store database for semantic chunk retrieval.
- Local model connection support via Ollama.
- AI study tools: Summary Generator, Quiz Generator, Flashcard Generator, ask-doc Chatbot.
- Metrics & Study Analytics tracking dashboard.
- Custom Streamlit layout and CSS theme styling.
